# AgomTUI Development Docs

这个目录是 AgomTUI 的开发文档入口。根目录 README 负责产品介绍和快速启动；开发规则、架构边界、测试和 CI 护栏在这里维护。

## 阅读路径

- 想了解项目边界：先看 [architecture/product-split.md](architecture/product-split.md)，再看 [architecture/compiler-architecture.md](architecture/compiler-architecture.md)。
- 想开始开发：看 [development/README.md](development/README.md)。
- 想确认代码规则：看 [development/development-standards.md](development/development-standards.md)。
- 想跑本地校验：看 [development/testing.md](development/testing.md)。
- 想理解 PR 护栏：看 [development/ci-guardrails.md](development/ci-guardrails.md)。
- 想跟踪拆分和迁移阶段：看 [migration/migration-plan.md](migration/migration-plan.md)。

## 目录结构

- `architecture/`：产品边界、compiler 架构、上游提取说明。
- `development/`：开发指南、开发标准、测试命令、CI 护栏。
- `migration/`：迁移计划和阶段路线。
- `assets/`：README 和 demo 使用的截图资源。
